Overview

서비스 개요

S&P는 식단 관리, 식수 예측, 식자재 재고, 발주 흐름을 연결해 급식 운영 업무를 지원하는 프로토타입 시스템입니다.

대시보드

대시보드는 월간 식단 캘린더를 중심으로 선택 날짜의 식단, 예측 식수, 부족 재고, 발주 대기 상태를 한 화면에서 확인하는 공간입니다.

식단 관리

날짜별 조식, 중식, 석식 메뉴를 등록하고 수정합니다. 등록된 식단은 예측 식수 확인과 식단기반 스마트 발주 흐름에 연결됩니다.

식수 예측

기준일을 선택하면 조식, 중식, 석식의 예측 식수를 생성하고 확인할 수 있습니다. 예측 결과는 대시보드와 예측 캘린더에서 확인할 수 있습니다.

식자재/재고

식자재 목록과 입출고 기록을 기준으로 현재고를 계산합니다. 안전재고 이하 품목은 대시보드의 부족 재고 영역에 표시됩니다.

발주 관리

발주 관리는 일반 발주와 식단기반 스마트 발주 흐름으로 구성됩니다. 식단기반 스마트 발주는 등록된 식단과 레시피를 바탕으로 필요한 식재료를 계산합니다.

FAQ

예측 식수가 보이지 않아요.

해당 날짜의 예측을 먼저 실행해야 대시보드에 예측 결과가 표시됩니다.

캘린더에 식단 없음으로 표시돼요.

해당 날짜에 식단이 등록되지 않은 상태입니다. 식단 등록/수정 버튼을 통해 식단을 추가할 수 있습니다.

부족 재고는 어떤 기준인가요?

입고, 출고, 폐기, 조정 로그를 합산해 계산한 현재고가 안전재고 이하인 경우 부족 재고로 표시합니다.
